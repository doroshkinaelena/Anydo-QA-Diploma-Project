# Diploma Project: Any.do Mobile App Testing 📱

## 🎯 Project Overview (EN)
This is my graduation project focused on comprehensive testing of the Any.do mobile application. Completed as part of my transition to a QA Engineer role, it covers the full Quality Assurance lifecycle.

### 📊 Key Metrics:
* 80+ checks (combination of checklists and detailed test cases).
* 3 bugs identified (Blocker, Major, Minor), reported according to industry standards.
* 11 devices in the device coverage matrix (iOS & Android).

### 🛠 Skills Applied:
* Requirements Analysis: Decomposition of Any.do features (tasks, subscriptions, calendar).
* Test Design: Boundary Value Analysis, Equivalence Partitioning, Error Guessing.
* Testing Types: Functional, UI/UX, Interruption (calls, network loss), Regression, and Installation testing.
* Data Sync: Verified data exchange between Mobile and Web versions.

## 🎯 О проекте (RU)
Это мой дипломный проект, посвященный комплексному тестированию мобильного приложения Any.do. Работа выполнена в рамках перехода в профессию QA Engineer и охватывает полный цикл обеспечения качества.

### 📊 Ключевые показатели:
* 80+ проверок (комбинация чек-листов и детальных тест-кейсов).
* 3 найденных бага (Blocker, Major, Minor), оформленных по стандартам.
* 11 устройств в матрице покрытия (iOS и Android).

### 🛠 Что реализовано в работе:
* Анализ требований: Декомпозиция функций Any.do (задачи, подписки, календарь).
* Тест-дизайн: Применение техник (граничные значения, эквивалентные классы, предугадывание ошибки).
* Виды тестов: Функциональное, UI/UX, тестирование прерываний, регрессионное и инсталляционное тестирование.
* Синхронизация: Проверка обмена данными между мобильной и Web-версиями.
